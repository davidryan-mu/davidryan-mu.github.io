Hiragana Tutor App

A browser-based single page application that assists the learning of Hiragana, one of the three character sets used in the Japanese language. The app provides functionality for saving and reviewing learner progress using AJAX calls to an accompanying database via a web service created in PHP.

In Action

The video to the right displays the application being used. A user can select a training set, i.e. the characters they will be quizzed on. Using Javascript, the app then generates a quiz based on the training set.
The user can then sign up or log in to save their progress, which can be viewed in the progress tab.
