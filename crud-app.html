CRUD Book Database App

A simple browser-based single page application, developed in PHP, that facilitates CRUD (Create, Retrieve, Update, Delete) management of a single database table created with mySQL.

In Action

The video to the right displays the application being used. A user can add new records to the database with information on a particular book. They can update any column in a record by using the Book ID as a key. Records can be deleted by any column and allows for the deletion of mulitple records with the same info. The database can also be viewed at any time in a generated table.
